## Update Session Tag

Updates or adds a tag to a specific session. Works for both active and closed sessions.

### Parameters

<ParamField path="session_id" type="string" required>
  The unique identifier of the session to update
</ParamField>

<ParamField query="tag" type="string" required>
  The new tag value to assign to the session
</ParamField>

### Example Request

```bash
curl -X PUT "http://localhost:8000/api/sessions/abc123xyz/tag?tag=NewTag"
```

### Example Response

```json
{
  "session_id": "abc123xyz",
  "container_id": "abc123xyz789",
  "vnc_port": 5002,
  "created_at": "2024-12-14T10:30:00Z",
  "env_type": "browser",
  "debug_port": 4002,
  "status": "running",
  "tag": "NewTag"
}
```

### Error Responses

| Status Code | Description |
|------------|-------------|
| 404 | Session not found |
| 500 | Failed to update tag |

### Notes

- Tags can be updated for both active and closed sessions
- Updating a tag does not affect any other session properties
- Tags are optional but useful for session organization
