# Embedding Live Sessions

You can embed MarinaBox sessions in your web applications using an iframe. This allows for seamless integration of live session views into your own interfaces.

## Basic Embedding

Here's a simple example of embedding a session:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Embedded Session View</title>
    <style>
        .vnc-iframe {
            position: fixed;
            top: 0;
            right: 0;
            width: 70%;
            height: 70%;
            border: none;
            z-index: 1000;
            box-shadow: 0 0 10px rgba(0,0,0,0.2);
        }
    </style>
</head>
<body>
    <iframe 
        class="vnc-iframe"
        src="http://127.0.0.1:5004/vnc.html?resize=scale"
        allow="clipboard-read; clipboard-write"
    ></iframe>
</body>
</html>
```

## URL Parameters

Customize the embedded view with these URL parameters:

- `resize=scale`: Scales the view to fit the iframe size
- `autoconnect=true`: Automatically connects to the VNC session
- `view_only=true`: Disables user interaction with the session

Example with all parameters:
```html
<iframe src="http://127.0.0.1:5004/vnc.html?resize=scale&autoconnect=true&view_only=true"></iframe>
```

## Responsive Design

For a responsive embed that maintains aspect ratio:

```html
<style>
    .vnc-container {
        position: relative;
        width: 100%;
        padding-bottom: 56.25%; /* 16:9 aspect ratio */
    }
    .vnc-iframe {
        position: absolute;
        top: 0;
        left: 0;
        width: 100%;
        height: 100%;
        border: none;
    }
</style>

<div class="vnc-container">
    <iframe 
        class="vnc-iframe"
        src="http://127.0.0.1:5004/vnc.html?resize=scale"
        allow="clipboard-read; clipboard-write"
    ></iframe>
</div>
```
